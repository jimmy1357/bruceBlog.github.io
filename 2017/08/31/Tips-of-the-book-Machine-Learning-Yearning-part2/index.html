Tips of the Book "Machine Learning Yearning" ( Part Ⅱ )

Machine Learning
  • Machine Learning Yearning

PART Ⅱ Basic Error Analysis

1.观察你的dev set 并评估优化点

在你的喵星人分类器中，有一些汪星人也被分类成喵星人了。

这时，你是否需要花大量时间来优化模型，以避免这些汪星人被误分类呢？使用下面的方法，帮助你做决策：

(1) 选一些（例如100个）分类错误的图片。

(2) 人工检查这些图片，计算其中被分类为猫的狗的图片在总分类错误图片中的占比。

如果在这些错误分类的图片中，汪星人只占5%。而此时你的分类器已经有90%的准确度了。那么，优化汪星人带来的效应，最高只能提升你的猫咪分类器 5% * 10% = 0.5%。 也就是说，优化完汪星人，你的喵星人分类器准确度最多达到90.5%，看起来意义不大。

相反，如果如果在这些错误分类的图片中，汪星人占50%，那优化完毕以后，你将最高提升模型准确率到95%，这样的话，优化汪星人还是很有必要的。

错误分析可以帮助你确定哪个优化方向更有价值。

2.在错误分析中评估多个优化点

在你的喵星人分类器中，有多个误分类情况：把狮子，黑豹误分类为猫；把狗误分类为猫；模糊图像的识别率低……

如何抉择优化顺序？

画一个表，选一定数量的误分类图片，记录他们被误分类的原因以及 被误分类的图像的特征…在这个过程中你可能会发现新的误分类原因及特征。（例如，经过一段时间的图片检查，你发现很多被误分类的图片都加了ins滤镜，这时你就可以添加一列“ins滤镜”，作为一个新的问题项）



在上述过程中，你可以想想，作为人类的一员，你是如何判断并正确的标记这些图片的？这对你提出新的问题项、解决这些问题项有很大的帮助。

误差分析是一个迭代的过程。

在检查完这100个误分类的图片后，你可能发现了一些新的问题项，这时，你需要重新检查一遍所有误分类的图片，看看他们是否存在新问题项中的问题。

在你完全检查完图片后，你得到如下的一张表：



你会发现，解决模糊问题带来的误分类误差，以及狮子，黑豹等的误分类是比较重要的。

当然，这并不意味着你必须首先矫正模糊问题或者狮子黑豹的问题，你还需要评估解决这些问题所花费的时间与你项目预期的时间，以及你的人力分配和预算。
